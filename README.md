# Tmax4 (من أجل تبارك ❤️)

تطبيق خاص يعمل كمرافق عاطفي ذكي.

## طريقة الرفع على GitHub وتشغيله (Deploy)

للحفاظ على أمان التطبيق وتشغيله بشكل صحيح على GitHub Pages، اتبع الخطوات التالية بدقة:

1. **ارفع الكود**: قم برفع الملفات إلى مستودع GitHub (Repository) باسم `Tmax4`.
2. **إعداد المفتاح السري (API Key)**:
   - اذهب إلى صفحة المستودع على GitHub.
   - اضغط على **Settings** (الإعدادات) في الشريط العلوي.
   - من القائمة الجانبية اليسرى، اختر **Secrets and variables** ثم **Actions**.
   - اضغط على زر **New repository secret** (باللون الأخضر).
   - في خانة **Name** اكتب: `GEMINI_API_KEY`
   - في خانة **Secret** الصق مفتاح Gemini API الخاص بك.
   - اضغط **Add secret**.
3. **تفعيل النشر**:
   - اذهب إلى تبويب **Actions** في الأعلى، وتأكد أن الـ Workflows مفعلة.
   - سيقوم GitHub تلقائياً ببناء التطبيق ونشره (قد يستغرق دقيقة أو دقيقتين).
   - اذهب إلى **Settings** > **Pages**.
   - ستجد رابط التطبيق المنشور هناك (مثال: `https://aws203.github.io/Tmax4`).

**ملاحظة هامة**:
التطبيق مصمم ليعمل "بدون خادم" (Serverless)، ويتم حقن المفتاح بشكل آمن أثناء عملية النشر فقط، ولا يظهر في ملفات الكود المفتوحة.
